Popular Talks
Efficiently reduce toxicity of your large-language models using parameter-efficient fine-tuning (PEFT) and reinforcement learning from human feedback (RLHF)
Toxicity and hate speech have no place in modern society. We can use reinforcement learning (RL) to reduce the toxicity of our LLMs, however RL is a compute-intensive and expensive process that typically requires many GPUs across a distributed cluster. In this talk, I will discuss and demonstrate how to combine parameter efficient fine tuning (PEFT) with reinforcement learning from human feedback (RLHF) for lightweight detoxification of an LLM on a single GPU. The talk will feature a live demo to demonstrate the detoxification process in a cost-effective manner. The code is all public and the github repo will be shared.

Generative AI and Machine Learning
Orchestrating a fine-tuning pipeline for generative large-language models using Amazon SageMaker
In this talk, we build an end-to-end pipeline to fine-tune and deploy a generative large-language model (LLM) using Amazon SageMaker. The pipeline includes feature engineering, supervised-fine tuning (SFT), parameter-efficient fine-tuning (PEFT), model evaluation, and model deployment. We also discuss mechanisms to build a human-feedback workflow to further fine-tune and improve our model. We show how to incorporate the human feedback back into the fine-tuning pipeline with reinforcement learning (RLHF). 

Generative AI and Machine Learning
Scaling AI and machine learning workloads with Ray on AWS, Kubernetes, and BERT
Open source Ray is great for modern high-performance data processing and machine learning workloads.  In this talk, I will explain distributed Ray - and how it differs from Dask and Apache Spark.  I will also discuss various tips and tricks to optimizing Ray on AWS and Kubernetes.  Next, I will highlight some AWS key contributions to the open source Ray project to improve performance, scalability, and operational efficiency.  Lastly, I conclude the talk with a demo using Ray's AI Runtime (AIR) to train and serve a large, distributed BERT model with Amazon Elastic Kubernetes Services (EKS).  This talk is based on this blog post:  https://aws.amazon.com/blogs/opensource/scaling-ai-and-machine-learning-workloads-with-ray-on-aws/

Smokey and the Multi-Armed Bandit featuring BERT Reynolds and Reinforcement Learning
